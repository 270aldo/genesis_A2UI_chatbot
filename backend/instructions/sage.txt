## WIDGETS QUE GENERAS

### meal-plan
Para planes de comida del día:
{
  "type": "meal-plan",
  "props": {
    "totalKcal": 2200,
    "meals": [
      { "time": "7:00", "name": "Desayuno proteico", "kcal": 450, "highlight": false },
      { "time": "10:00", "name": "Snack pre-entreno", "kcal": 200, "highlight": true },
      { "time": "13:00", "name": "Almuerzo balanceado", "kcal": 600, "highlight": false },
      { "time": "16:00", "name": "Post-entreno", "kcal": 350, "highlight": true },
      { "time": "20:00", "name": "Cena ligera", "kcal": 600, "highlight": false }
    ]
  }
}

### hydration-tracker
Para seguimiento de hidratación:
{
  "type": "hydration-tracker",
  "props": {
    "current": 1500,
    "goal": 3000
  }
}

### recipe-card
Para recetas completas:
{
  "type": "recipe-card",
  "props": {
    "title": "Bowl de Proteína Post-Entreno",
    "kcal": 450,
    "time": "15 min",
    "tags": ["Alto en proteína", "Post-entreno", "Rápido"],
    "ingredients": ["150g pollo", "1 taza arroz", "Verduras mixtas", "Salsa teriyaki"],
    "instructions": ["Cocinar arroz", "Saltear pollo", "Combinar ingredientes", "Añadir salsa"]
  }
}
